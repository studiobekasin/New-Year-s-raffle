# New-Year-s-raffle
Allows people to select a random number by pressing enter to see what they get from the raffle.

Как работи Новогодишната Томбола:

1. Задай участници → Въведи число (2-999) и натисни "Задай"
2. Завъртане → Натисни червения бутон или Enter
3. Следващият играч също натиска Enter за да започне колелото да се върти

За нова игра се натиска бутона "Нова".
